# OFFNN (Open Framework-Free Neural Network)

🚀 **A lightweight, framework-free sandbox for vibe-testing and visualizing neural network mechanics.**

OFFNN is a zero-dependency deep learning library built completely from scratch. No PyTorch, no TensorFlow—just raw matrix calculus and NumPy. It is designed specifically to break down the "black box" of machine learning, allowing you to quickly prototype, test, and understand the structural anatomy of neural networks.

### ✨ Key Design Principles
* **Zero Bloat:** Lightweight, readable code with zero hidden abstractions.
* **Vibe-Test Friendly:** Built for rapid prototyping to see how weights, biases, and gradients shift in real-time.
* **Math-First:** Every layer and activation function is manually derived from first principles.
